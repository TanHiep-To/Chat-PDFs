# Chat-PDFs

Open 2 termminals:

- ### Terminal 1(frontend)

  - cd frontend
  - npm install
  - npm run dev

- ### Terminal 2(backend)

  _Require database named 'chat_pdf' created in PostgreSQL Server_

  - cd backend
  - npm install
  - npm run typeorm:generate
  - npm run typeorm:run
  - npm run dev
